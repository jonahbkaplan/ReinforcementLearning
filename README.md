# Run the following commands to create the environment and install dependencies:

1. Create virtual environment
```
python -m venv venv
```

2. Activate environment
Windows:
```
.\venv\Scripts\activate
```
Mac/Linux:
```
source venv/bin/activate
```

3. Install requirements
```
pip install -r requirements.txt
```


# Instruction for how to run the RL algorithms:

All source code related to each algorithm is in the corresponding directory in algorithms folder. In order to run each algorithm, open and run the test{Algorithm}.ipynb file.
